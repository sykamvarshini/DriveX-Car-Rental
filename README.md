# 🚗 DriveX - Premium Car Rental Platform

DriveX is a full-stack MERN car rental web application built with React, Node.js, Express, and MongoDB.

The platform allows users to browse premium vehicles, book cars, explore categories, and manage rentals through a modern responsive interface.

---

# ✨ Features

## 👤 User Features
- User Registration & Login
- JWT Authentication
- Browse Available Cars
- Search Cars
- Category-Based Filtering
- Car Details Page
- Book Cars
- View Booking History
- Responsive Modern UI

---

## 🛠️ Admin Features
- Admin Dashboard
- Add New Cars
- Manage Car Listings
- Dynamic MongoDB Data Storage

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/your-username/drivex.git
```

---

# Install Frontend Dependencies

```bash
cd client
npm install
```

---

# Install Backend Dependencies

```bash
cd server
npm install
```

---

# ▶️ Run Frontend

```bash
cd client
npm start
```

---

# ▶️ Run Backend

```bash
cd server
npm run dev
```

---

# 🌐 Environment Variables

Create a `.env` file inside `server/`

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
